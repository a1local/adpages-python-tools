# Privacy

AdPages Tools for Python is a local utility package.

- No external API calls.
- No hidden tracking.
- No hosted backend.
- No telemetry.
- No cookies.
- No account system.
- No data is sent to AdPages or any third party by this package.

The CLI reads the values supplied through command-line arguments and prints results to standard output. The Python modules return values in memory to the calling code.

Users are responsible for handling any campaign URLs, business details, and advertising copy they pass to the package.
